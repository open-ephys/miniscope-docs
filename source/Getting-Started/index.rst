#################
Getting Started
#################

You can follow the :ref:`quickstartguide` to learn how to start collecting data right away. However, we recommend reading the :ref:`hardwareguide` page carefully if you're going to use our hardware, and especially if you're going to make any modifications to it. When something goes wrong, or if your data look suspicious, knowing the basics of how the hardware works makes it easier to troubleshoot.
