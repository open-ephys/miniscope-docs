#########
Commutate
#########

..  image:: /_static/images/uclaminiscopev4-miniscopedaq-commutate.svg
    :alt:   image of workflow
    :width: 50%
    :align: center

After following this tutorial, the user will be able to:

*   automatically rotate the coaxial cable when the UCLA Miniscope v4 rotates 

*   manually rotate the commutator:

    *   clockwise when the user presses `Alt + Left` keys on their keyboard

    *   counter-clockwise when the user presses `Alt + Right` keys on their keyboard
